# Netflix-Clone-UI
 A responsive Netflix-inspired web application built to replicate the user interface and user experience of Netflix. This project demonstrates front-end development skills, responsive design principles, and modern web technologies.

 #  🚀Features
  Netflix-inspired UI design
  Responsive layout for desktop, tablet, and mobile devices
  Interactive navigation bar
  Movie/TV show showcase sections
  Hover effects and smooth animations
  Modern and clean user interface

 #  🛠️ Technologies Used
  HTML5
  CSS3
  JavaScript
